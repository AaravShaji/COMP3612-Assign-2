About Clothify

Clothify is a modern, student-built online clothing store designed to showcase clean UI, responsive layouts, JSON-powered data rendering, and single-page application navigation. Built as part of COMP 3612 ‚Äì Web Development, the website demonstrates practical front-end engineering skills through dynamic interfaces and real-time content updates.

Our Mission

Our mission is to create an elegant and intuitive shopping experience that feels fast, visually appealing, and easy to navigate. Every product is loaded directly from JSON data, ensuring a modular design and a clean separation of content from layout. We aim to mimic the structure of real e-commerce sites while maintaining readable and maintainable code.

Why Clothify?

  • Fully client-side SPA navigation with animated view transitions
  • Dynamic product loading using JSON and JavaScript
  • Clean component-based layout (header, footer, modals, views)
  • Modern UI elements including gradients, modals, and animations
  • Accessible design following good HTML semantics

The Developers

Clothify was created by Aarav Shaji and James Yap, Computer Science students at Mount Royal University. With shared interests in software engineering, UI/UX design, and game development, our goal was to build something functional, polished, and academically strong.
